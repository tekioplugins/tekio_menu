
# Base Menu Builder
This is the base json necessary to create the menu.

  

## Fields

  

You can edit the following fields:

  
- menuKey - Menu key.
- homeListItems **(required)** - This is a list of items that will consist on the menu buttons.



  

## Json Format

  

    {
       "menuKey":"example_key",
       "homeListItems":[
          // Add items here
       ]
    }
