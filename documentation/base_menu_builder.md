# Base Menu Builder

This is the base json necessary to create the menu.

## Fields

You can edit the following fields:

- menuKey - Menu key.
- menuListButtons **(required)** - This is a list of items that will consist on the menu buttons.

## Json Format

  

    {
       "menuKey":"example_key",
       "menuListButtons":[
          // Add items here
       ]
    }
